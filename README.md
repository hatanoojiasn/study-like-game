# ゲームっぽい学習ラボ

このプロジェクトは、**壊れた世界を修理しながら学ぶ**ための学習ゲームです。
単なる作業タスクではなく、観測→仮説→修復→世界改善のループを回して、OS・コンパイラ/VM・ゲームエンジン・数学/物理を横断的に習得します。

## 最終的にできるようになること
- 同じゲームロジックを Host 環境から自作OS想定環境へ段階移植できる。
- 字句解析からVM実行までを最小構成で作り、敵AIへ適用できる。
- 入力/描画/時間/タスク/物理を、測定可能な指標で改善できる。

## 学び方（ゲームループ）
1. `docs/quests/` のミッションを1つ選ぶ。
2. 「症状」を再現してログを取り、原因仮説を立てる。
3. 公開APIを増やさずに内部実装で修理する。
4. 受け入れ条件を満たし、報酬（アンロック）を獲得する。
5. 次Questへ進み、Tierを登る。

## クエストの遊び方
- 難易度と所要時間を `docs/roadmap.md` で確認する。
- 詰まったら各Questの「失敗例」と「用語ミニ辞典」を先に読む。
- 進捗はHUDメトリクス（FPS/dtジッタ/テレメトリ誤差/命令数）で可視化する。

## セットアップと実行
環境構築・ビルド・実行手順は `docs/README.md` を参照してください。
